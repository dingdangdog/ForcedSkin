# 021 站点白名单禁用主题机制

时间：2026-04-27

## 需求

加入白名单机制：命中白名单的网站，插件主题不生效，永远保持网站原始效果。

## 方案

1. 新增存储键
   - `siteWhitelist`（`chrome.storage.sync`）
2. Popup 增加白名单配置
   - 新增多行输入框（每行一个域名）
   - 新增“保存白名单”按钮
3. Background 下发统一设置
   - 将原先仅下发 `themeMode` 扩展为下发 `mode + whitelist`
   - 新增消息：`GET_SETTINGS`、`SET_WHITELIST`
4. Content 执行白名单判定
   - 规范化域名（去协议、去 `www.`、去路径、小写）
   - 当前 hostname 与白名单项“完全匹配或子域匹配”时，直接 `removeTheme()`

## 结果预期

白名单中的站点及其子域名将始终保持网站原生样式，不再被插件主题覆盖；非白名单站点仍按当前主题模式正常生效。
