# 005 B站回归修复记录

时间：2026-04-27

## 问题

在“频道区图标/文字跟随主题”修复后，出现了 B站其他区域主题生效减弱的回归现象。

## 修复策略

采用“只补充不削弱”原则：

1. 保留频道区专项修复逻辑
2. 新增 B站全局兜底样式层，恢复主页/视频页/评论区/侧栏等核心区域的稳定生效
3. 保持图片/视频/蒙版等高风险元素跳过策略不变，避免遮罩异常

## 代码改动

- 更新 `src/content/content.js`
  - `cssForRoot` 新增 `adapterId` 参数
  - 当适配器为 `bilibili` 时注入额外 fallback CSS
  - `applyMode` 调整为先解析适配器，再生成根样式

## 结果

频道区修复继续有效，同时恢复其他 B站区域的主题覆盖效果，降低回归风险。
