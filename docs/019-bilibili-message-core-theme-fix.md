# 019 B站消息页核心区域主题修复

时间：2026-04-27

## 问题

B站消息页（`bilibili-message`）主要区域未完整应用暗色主题，出现背景与内容区色彩不一致。

## 修复

在 bilibili 适配器中增加消息页专项：

1. 主容器与内容区
   - `message-bgc/message-bg/message-main/message-content`
   - 会话区、聊天区、输入区、系统消息区、私信区、at/回复列表
   - `im-*` 容器族（用于不同构建版本）
2. 次级组件与激活态
   - 菜单项、Tab 项、会话激活项、工具栏、输入包裹、列表项、卡片

## 结果预期

消息页核心区域在暗色模式下背景、边框、文字统一，避免“主内容区仍是站点默认浅色”。
