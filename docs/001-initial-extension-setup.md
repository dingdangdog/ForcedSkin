# 001 初始扩展实现记录

时间：2026-04-27

## 变更目标

从空目录初始化一个 Chrome 插件，实现网页统一主题的基础能力，支持三种状态：

- 亮色主题
- 暗色主题
- 不修改

## 本次变更内容

1. 新增 `manifest.json`
   - 使用 Manifest V3
   - 声明 `storage`、`scripting`、`tabs` 权限
   - 注入内容脚本至所有网站
2. 新增后台脚本 `src/background.js`
   - 保存与读取主题状态（`chrome.storage.sync`）
   - 处理弹窗发来的主题切换消息
   - 广播主题到所有已打开标签页
   - 在标签页切换和加载时重新同步主题
3. 新增内容脚本 `src/content/content.js`
   - 读取主题状态并注入覆盖样式
   - 支持亮色、暗色、不修改三种模式
   - 仅覆盖颜色相关样式，不改页面布局结构
4. 新增弹窗页面
   - `src/popup/popup.html`
   - `src/popup/popup.css`
   - `src/popup/popup.js`
   - 提供三种主题状态切换并显示当前状态

## 说明

该实现为基础版本，目标是“尽可能统一颜色风格”。由于网页样式差异极大，个别站点可能出现局部不一致，后续可以通过更细粒度规则持续优化。
