# 044 扩展弹窗 Tab 布局与信息架构重设计

## 日期

2026-04-30

## 变更摘要

对 `extension/src/popup` 做信息架构与视觉重构（版本 **0.2.2**），不改变既有后台消息与主题同步逻辑。

### 布局

1. **顶栏**：左侧品牌（Logo + ForcedSkin）；右侧 **头像 + 昵称**（未登录显示「游客」与首字母占位）、**打开官网** 图标按钮。顶栏 **不包含** 退出登录。
2. **Tab 三页**
   - **主题**：模式选择（亮/暗/关）+ 对应横向配色条 + 状态提示。
   - **白名单**：英文半角逗号 `,` 分隔的域名 **多行编辑**、「保存白名单」、当前站点快捷加入/移除。
   - **账号**：OAuth 登录；已登录则展示大头像、昵称、邮箱、**全部亮/暗可选主题**（可点击切换）、账号同步摘要、「同步主题」、底部 **退出登录**。
3. **样式**：360px 宽、分段卡片、圆角 Tab、选项选中描边、统一间距与字重。

### 文案与 i18n

新增 `_locales` 键：`tabThemes` / `tabWhitelist` / `tabAccount`、`guestNickname`、`openOfficialSite`、`whitelist*`、`panel*`、`overview*`、`accountSyncedTitle` 等（中英文）。

### 交互说明

- 白名单编辑器仅在点击「保存」时写入 `SET_WHITELIST`；解析规则为 **仅按英文逗号** 分割（与产品要求一致）。
- `chrome.tabs.create` 打开 `https://forcedskin.com`，需浏览器允许扩展创建标签页（MV3 默认具备）。
