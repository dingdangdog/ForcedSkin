# 022 Popup 当前网站白名单按钮化

时间：2026-04-27

## 问题

白名单依赖手动输入域名，操作繁琐，且无法直观看到“当前网站是否已在白名单”。

## 修复

1. Popup 改为当前网站驱动
   - 自动读取当前激活标签页 URL 并解析 hostname
   - 显示“当前网站”和“白名单状态”
2. 新增一键操作按钮
   - `加入白名单`
   - `移除白名单`
3. 按钮状态联动
   - 已在白名单：禁用“加入”，启用“移除”
   - 不在白名单：启用“加入”，禁用“移除”
4. 非 http/https 页面兜底
   - 无法识别当前网站时，显示不可操作并禁用按钮

## 结果预期

用户无需手动输入域名，直接在 Popup 一键将当前网站加入/移出白名单，交互更直观、准确。
