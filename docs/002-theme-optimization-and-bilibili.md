# 002 主题优化与 Bilibili 适配记录

时间：2026-04-27

## 变更目标

在已有“全局主题插件”基础上，解决主题覆盖效果粗糙问题，重点优化 `bilibili.com` 的显示一致性和可读性。

## 本次变更内容

1. 更新主题色板（亮绿 / 暗绿）
   - 亮色主题改为以 `#F8FFF8`、`#2C3E2C` 为核心的浅绿系
   - 暗色主题改为以 `#101410`、`#E0E0E0` 为核心的深绿系
   - 主色、强调色、边框色按提供配色方案映射到 CSS 变量
2. 调整样式注入策略
   - 从“全量强覆盖”改为“变量驱动 + 常见区域定向覆盖”
   - 删除 `body *` 级别的粗暴覆盖，减少站点原有视觉层次被破坏
   - 对表格、输入控件、面板、弹层、菜单等常见结构做稳定处理
3. 增加 Bilibili 专项样式补丁
   - 对主页/视频页/评论区/右侧栏等常见容器定向处理
   - 优化标题、副文本、统计信息、卡片阴影和边框统一性
4. 取消媒体统一滤镜
   - 移除对 `img/video/canvas/svg` 的亮度/对比度滤镜
   - 避免“整体发灰”与视频画面偏色问题

## 效果说明

本次优化重点是“看起来统一但不过度篡改”。不同网站结构差异很大，后续可继续针对高频站点补充细分规则，逐步提升一致性。
