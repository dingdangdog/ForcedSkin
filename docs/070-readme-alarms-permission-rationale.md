# 070 — readme：补充 `alarms` 权限商店说明

## 变更

- 在仓库根目录 `readme.md` 的「需请求权限的理由（英文分项）」中新增 **`alarms`** 小节。
- 说明与实现一致：`extension/src/background.js` 在用户选择自动主题模式时，通过 `chrome.alarms` 在本地日出/日落边界唤醒 service worker，向标签页广播有效主题，并在非自动模式下清除闹钟。

## 理由摘要（中文）

MV3 后台可被挂起；自动模式需在本地时间边界切换亮/暗主题，`alarms` 用于在下一边界单次唤醒并推送设置，不做无关定时任务。
